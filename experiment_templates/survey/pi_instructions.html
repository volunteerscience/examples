How to use:

  1. Create your survey in Qualtrics
  2. Add embedded data
    • Click the "Edit Survey" tab
    • Click "Survey Flow" in the menu above your survey
    • Click "+ Add a New Element Here"
    • Click "Embedded Data"
    • type "testId"
    • Click "Add a New Field"
    • type "subjectId"
    • Click "Save Flow" at bottom
  3. Launch Survey
    • Set the anonymous link in the custom variable "survey_link"
    • Optionally set a title (or leave it blank)
    • Set the survey height so it fits well in the screen (you'll need to test this). Try "500px"
  4. Test your survey by clicking "Launch"
